# [who-hosts-hate](https://who-hosts-hate.com)

Who hosts hate on the internet?

## Requirements

- Python 3.7

## Setup

PIPENV

```sh
python3 -m venv .venv
source .venv/bin/activate # or `source .venv/bin/activate.fish`
pip install -r requirements.txt
```

## Build

Generate `index.html` via `build.py`.

```
usage: build.py [-h]
```
